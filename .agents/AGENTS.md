# Reglas del Proyecto (Universal Video Downloader)

## Actualización Automática en GitHub
Cada vez que se realicen modificaciones, creaciones o eliminaciones de archivos de código o documentación como parte de una tarea:
1. Al finalizar y verificar los cambios, ejecuta automáticamente `git add .`.
2. Crea un commit con un mensaje claro, descriptivo y en español que resuma los cambios realizados (`git commit -m "..."`).
3. Sube los cambios al repositorio remoto mediante `git push` (o `git push origin main`).
4. Informa al usuario en el resumen final que los cambios ya han sido sincronizados y subidos a GitHub.
